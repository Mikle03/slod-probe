# Results

This directory contains real SciBERT probe metrics for the in-domain, exact-length-controlled, and external cross-domain conditions; confusion-matrix CSVs for the embedding probe and both baselines; extraction summaries; and concise experiment summaries. The completed cross-domain condition trains on QASPER NLP and tests without adaptation on a separately collected PMC BioC biomedical set. Because both domain and source corpus change, it is reported as combined domain-and-corpus transfer.
